# GitHub Pages

This repository keeps its documentation in `/docs` so it can be published with **GitHub Pages**.

## Recommended configuration

- **Source:** Deploy from a branch
- **Branch:** `main`
- **Folder:** `/docs`

## Notes

- The documentation is intentionally lightweight and points readers back to the canonical schema READMEs under `/credentials/*`.
- External links are expected to be validated in CI in downstream repos; this repo focuses on deterministic schema validation.


- AIS-1 schema profile landing page: `docs/ais1-schema-profile.md`
