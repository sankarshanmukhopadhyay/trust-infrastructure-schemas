# Release v0.5.0

This release reframes the repository as an **Open Trust Artifact Model** for operational digital trust infrastructure.

## Highlights

- introduces an explicit artifact-model narrative
- adds a documented artifact taxonomy
- adds machine-readable taxonomy metadata under `model/`
- updates architecture and documentation to position the repository as the trust-artifact layer between identity primitives and governance systems
- removes archive cruft such as `.DS_Store`
