# v0.6.0 — Open Trust Artifact Model and OASF/AIS-1 Composition Readiness

**Release date:** 2026-04-07

## Summary

`v0.6.0` repositioned the repository as an Open Trust Artifact Model for operational trust infrastructure. The release added OASF publication and evaluation artifacts, ODRL policy-reference support, AIS-1 experimental profile material, reusable authority-boundary and artifact-reference contracts, and composition examples connecting declarations, evidence, evaluations, and registry publication.

## Why this release mattered

The repository moved beyond credential schema publication into reusable trust-infrastructure artifacts. It started to provide the shared shapes needed by assurance, conformance, registry, and baseline repositories.

## Added

- OASF publication profile schema.
- OASF evaluation envelope schema.
- OASF control crosswalk mapping.
- ODRL policy reference schema and sample.
- AIS-1 experimental bonded-agent profile.
- Reusable artifact reference schema.
- Reusable authority boundary schema.
- Cross-repo composition examples.

## Changed

- README and model documentation were updated to frame the repository as an Open Trust Artifact Model.
- Registry examples were expanded with OASF-aware and ODRL-aware references.

## Compatibility

This release was intended as an additive minor release. It introduced new artifact families without removing existing credential, assurance, conformance, control, or registry schemas.
