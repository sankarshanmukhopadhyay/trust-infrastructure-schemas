# Trust Infrastructure Schemas v0.12.0

This release promotes TIS to Candidate after repairing clean-checkout validation, adding a machine-readable status contract, moving CI to Node 24, publishing the complete documentation with Just The Docs, adding architecture and lifecycle diagrams, and producing Candidate evidence.

## Breaking changes

No intentional schema semantic break is introduced. Validation dependencies and workflow requirements are now explicit and reproducible.

## Assurance boundary

Candidate is not certification. Independent interoperability testing remains incomplete.
