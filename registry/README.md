# Registry artifacts

This directory contains non-normative registry-state schemas and examples that show how trust artifacts may be indexed inside an ecosystem registry.


Registry entries may optionally carry OASF publication profile, evaluation envelope, evidence, and semantic-binding references when a downstream ecosystem publishes those artifacts.

Registry entries may also carry AIS-1-oriented trust signals, such as recognized issuers, sponsor authorities, required tier thresholds, and policy notes clarifying that bonded identity does not by itself authorize consequential actions.
